# 宜启智控Pro官网

宜启智控Pro是一款智能物联网自动化控制器，旨在消除自动化控制和物联网应用的技术门槛，通过提供无需编程的解决方案，让用户能够轻松实现复杂的自动化逻辑配置。

## 项目概述

"宜启智控Pro"官方网站是一个集产品展示、技术文档、学习教程于一体的综合性平台，服务于开发者、工程师和终端用户。该平台的目标是成为全面的技术文档和学习资源中心，帮助用户快速掌握"宜启智控Pro"的使用方法。

## 核心功能

- **产品展示**：展示不同型号的宜启智控Pro产品及其规格
- **应用场景**：提供智慧农业、智慧楼宇、工业控制、教学实训等领域的应用案例
- **技术文档**：提供详细的硬件接口说明和MQTT协议文档
- **学习教程**：包含43节课程的完整教程系统，覆盖从基础到高级的所有功能
- **多平台对接**：支持Home Assistant、ThingsBoard、涂鸦生态等多种平台

## 技术特点

- **无需编程**：通过手机APP即可配置自动化逻辑，小白也能快速上手
- **广泛兼容**：支持市面上95%的传感器，兼容各种信号类型
- **多通信方式**：支持WiFi/4G/蓝牙等多种联网方式
- **强大接口**：包含继电器输出、PWM输出、开关量输入、模拟量输入、RS485 Modbus、IIC等接口
- **即插即用**：快速部署，无需复杂的安装配置

## 技术架构

- **框架**：[Astro](https://astro.build/) v5.16.6
- **样式**：[TailwindCSS](https://tailwindcss.com/) v4.1.18
- **语言**：TypeScript
- **构建工具**：Vite

## 安装与运行

1. 克隆项目
   ```bash
   git clone <repository-url>
   cd ylai_web
   ```

2. 安装依赖
   ```bash
   npm install
   ```

3. 启动开发服务器
   ```bash
   npm run dev
   ```

4. 构建生产版本
   ```bash
   npm run build
   ```
